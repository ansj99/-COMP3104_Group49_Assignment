##Information about the COMP 3104 DevOps course.##

DevOps Course Overview
DevOps is a collaborative approach that combines software development (Dev) and
 IT operations (Ops). It emphasizes automation, collaboration, and integration to 
 help organizations rapidly deliver high-quality software while ensuring the stability 
 and security of their deployments. The goal of DevOps is to shorten the development
   lifecycle and provide continuous delivery with high software quality.

   Key Topics Covered in a DevOps Course
Project Management Methodologies

Agile Methodology: Frameworks like Scrum and Kanban.
Waterfall vs Agile: Differences in planning and execution.
Lean Practices: Improving efficiency and reducing waste in processes.
Software Architecture Concepts

Microservices Architecture: Breaking down applications into small, independent services.
Monolithic vs Microservices: Pros and cons of each.
Service-Oriented Architecture (SOA): Design principles for creating scalable systems.
Infrastructure as Code (IaC): Managing infrastructure through code rather than manual processes.
Key DevOps Tools

Version Control Systems: Git, GitHub, Bitbucket.
Continuous Integration/Continuous Deployment (CI/CD) Tools: Jenkins, CircleCI, Travis CI, GitLab CI.
Configuration Management: Ansible, Puppet, Chef, SaltStack.
Containerization: Docker, Kubernetes, Helm.
Cloud Platforms: AWS, Google Cloud, Azure for deploying scalable applications.
Monitoring & Logging: Prometheus, Grafana, ELK Stack (Elasticsearch, Logstash, Kibana), Splunk.
Automation & Orchestration: Terraform, AWS CloudFormation, Kubernetes.
